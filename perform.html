Ok, dieses Herzchen ist natürlich wunderschön ...



... aber warum ist es gerade 64x48 Pixel groß?


Die Auflösung dieser spannenden Frage:
  • Man nehme CAN Utils vom SocketCAN git Repository (runterladen und 'make' sagen) - alternativ unter Debian/Ubuntu: apt-get install can-utils
  • Erzeuge mindestens einen virtuellen CAN Bus vcan0 unter Linux 2.6.25+
    mit "ip link add dev vcan0 type vcan"
  • Lade diese CAN Logdatei wirheiraten.log herunter
  • Im ersten Terminal jetzt "canplayer -li -I wirheiraten.log" eintippen
  • Das zweite Terminal sollte wenigstens 55 Zeilen anzeigen können, also großziehen
  • Im zweiten Terminal jetzt "cansniffer -c -b vcan0" eingeben
  • Staunen :-)
Nun mag man sich fragen, wie kommt man als Tastentipper nur auf so eine bekloppte Idee?
